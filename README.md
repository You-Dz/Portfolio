# Portfolio

Portfolio personnel réalisé avec React + Node.js
lien domaine: https://portfolio-eight-beige-xjvnl7rg1t.vercel.app

## Installation

### Frontend

cd frontend
npm install
npm run dev

Variables d'environnement
VITE_API_URL=

#### Backend
cd backend
npm install
npm run dev

Variables d'environnement
MONGO_URI=
FRONTEND_URL=
EMAIL_ADMIN=
PASSWORD_ADMIN=
